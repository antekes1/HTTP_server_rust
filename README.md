# Building

<h5> In linux: </h5> 
Download the source code using:

```bash
git clone https://github.com/anetkes1/HTTP_server_rust
```

cd into repo directory and run:

```bash
cargo run
```
<h4>*Remember you must have installed rust and cargo </h5>
